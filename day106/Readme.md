​

⭐ 함수의 3요소에 대해 복습하자 !

1️⃣ 인자, 인수, input, 입력값, 매개변수, args, 파라미터, ...

2️⃣ 반환값, 리턴값, return, 출력값, 결과값, output, ...

3️⃣ 함수의 기능 == 함수명

​
